# Project 1

This is a project (project1.py) about which extracts the web page title, page body text (without html tags) and all the urls and links attached to the web page.
I also have added assignment1.py which computes the count of common bits in Simhashes of given two urls.

Python Libraries - BeautifulSoup, re, sys
